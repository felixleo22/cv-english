Projets scolaires

Mai 2020 - 2 semaines

Atelier LP CIASIE
GeoQuizz

GeoQuizz est constitué de 3 applications distinctes exploitant un backend constitué de 3 services exposant chacune 1 API REST développée en Node.js et partageant des données communes :

- Une application de jeu qui permet de d'effectuer des parties. Elle est développée avec Vue.js, Vuetify.

- Une application back-office qui permet de gérer les données nécessaires au jeu, en particulier pour ajouter et géolocaliser des photos dans une série, créer de nouvelles séries de photos, configurer les paramètres associés à une série de photos (carte, distance, niveaux de jeu). Elle est développée avec Vue.js, Vuetify.

- Une application mobile qui permet de prendre des photos et de les géolocaliser, pour ensuite les transférer sur le back-end. Elle est développée avec NativeScript.

- En complément, les applications utilisent un service externe pour le stockage et l'upload des médias.
